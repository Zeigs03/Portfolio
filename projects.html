Featured Projects

Remote Work Salary Analysis

This project analyzed remote job postings across multiple industries using a cleaned and normalized dataset. I focused on understanding salary patterns by job type, experience level, and geographic preference. I created visualizations in Python using Seaborn and Matplotlib, and provided actionable insights for job seekers and HR departments exploring remote compensation strategy.

Colorado Weather Forecasting

Utilizing historical weather data from Colorado, I built regression models to predict temperature and conditions. The project incorporated feature engineering, exploratory visualizations, and predictive modeling using Scikit-learn. The results supported planning use cases, including weekly forecasting and trend reporting for local businesses or agencies.

Retail Sales Forecasting

This project involved forecasting future sales using historical retail data. I used time series analysis and machine learning models (XGBoost, ARIMA) to predict demand. Key outcomes included optimized inventory recommendations and enhanced sales strategy for seasonal planning, backed by interactive charts and scenario modeling.

Avocado Price Trend Analysis

Using Spark and Hive on a big data platform, I analyzed price trends and consumption volumes for avocados across U.S. regions. The project explored the distinction between conventional and organic avocados, highlighted top-volume markets, and surfaced actionable insights for grocery pricing strategy and sourcing decisions.

Food Delivery Profitability Dashboard

I built a dynamic dashboard to evaluate delivery profitability using SQL, Python, and Power BI. It included filters for geography, delivery type, and time periods. Profit margins, cost breakdowns, and customer behavior trends were visualized to aid data-driven decisions in delivery logistics and pricing strategies.

Synthetic Data Generator for Retail

To support model prototyping and maintain data privacy, I created a Python tool to generate realistic retail transaction data. The generator simulates customer baskets, timestamps, and item-level detail, with built-in support for both classification and regression targets. Ideal for ML model development and A/B testing.

Remote Work Job Scraper

Developed a Python web scraper using BeautifulSoup and Requests to gather remote job listings from multiple job boards. I standardized and cleaned the data into a unified format suitable for salary analysis, role frequency, and industry demand exploration. The resulting dataset fed into the Remote Work Salary project.

Retail Staffing Efficiency Model

This project applied logistic regression to model under- and over-staffing risks based on sales, traffic, and seasonal indicators. I developed an internal tool to assist retail managers with labor optimization, aiming to align staffing levels with customer flow and reduce overtime costs.

Customer Segmentation for Grocery Store

Using clustering algorithms (K-Means and DBSCAN), I segmented grocery customers based on shopping frequency, basket size, and product diversity. The model revealed distinct customer profiles, which can support targeted promotions and loyalty program design.

Fraud Detection Pipeline Architecture

I designed an end-to-end fraud detection architecture using Apache NiFi, Kafka, Hive, and Spark. The solution ingests and processes real-time financial transactions, flags anomalies based on defined thresholds, and supports scalable alerting. This project demonstrates big data orchestration and real-time analytics.
